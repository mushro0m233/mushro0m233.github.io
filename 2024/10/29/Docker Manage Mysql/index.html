Docker Manage Mysql

Mysql 远程连接：

  1. 配置 WSL2 镜像网络
  2. 配置 .wslconfig，物理机中的 WSL 配文件：autoMemoryReclaim=disabled
  3. 配置 daemon.json，WSL 中的守护进程：”iptables”:false
  4. 配置 Mysql 端口映射，以防与本地 Mysql 端口冲突
  5. 配置 Mysql 容器端口，绑定 0.0.0.0
  6. 创建 Mysql 远程连接用户
  7. 配置 WSL2 防火墙，允许 Mysql 端口通过
  8. 测试远程连接
